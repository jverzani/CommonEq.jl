# CommonEq

Package to share names for equality and inequality different from `==`, `<=`, ....

Export `Eq`, `Lt`, `Le`, `Ge`, `Gt`, `Ne` and unicode equivalents `⩵` ( `\Equal[tab]`), `≪` (`\ll[tab]`), `≦` (`\leqq[tab]`), `≧` (`\geqq[tab]`), `≫` (`\gg[tab]`), `≶` (`\lessgtr[tab]` or `≷`, `\gtrless[tab]`)
